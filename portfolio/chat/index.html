카카오톡채팅분석

by. park chulwoong

소개

카카오톡 대화내보내기 기능을 이용한 카카오톡 채팅 저장, 분석, 뷰어개발

개발환경

- OS : OSX

- Tools : VS code, Kibana

- 개발인원 : 1명

- 개발기간 : 1개월(기획포함)

주요기술

- Elasticsearch 6.2.2, Kibana 6.2.2

- Go 1.9, Gorilla

- Shell script, curl

- Vue.js 2.9, vue-bootrap, Toast UI Chart

구조

Parser

- Go를 이용한 카카오톡 채팅 파서기능 구현

- 네이티브언어로 고성능 기대

- 쉬운문법으로 생산성 향상

- 크로스컴파일이 가능함

Store & Analyser

- Elasticsearch를 이용한 데이터 적제 & 분석

- 오픈소스이며 한글 형태소 분석 플러그인 제공

- 채팅과 같은 full text검색에 효율

- 간단한 curl명령어로 bulk insert기능 제공

Viewer

- Vue.js를 이용한 프론트엔드 개발

- Toast UI CHART를 이용한 UI 차트개발(by Naver)

- Go를 이용한 REST API 백엔드 서버 개발

Parser

원본 채팅로그

단순텍스트, 날짜도 한글 포맷!!

파싱된 결과

Elasticsearch데이터에 맞게 포맷팅 됨

Store & Analyser

인덱스 매핑정보

Arirang 한글 형태소 분석 플러그인을 사용한다.

Inserter

curl을 이용하여 간단하게 파싱된 데이터를 벌크로 집어넣는다.

결과

Viewer - 키워드별


채팅내역중 가장많이 사용한 키워드와 그 키워드를 사용한 채팅참여자 사용분포도를 나타내고 있다.

Viewer - 사용자별


- 사용자 채팅순위 & 채팅분포도

- 사용자별 키워드 top100 & 채팅 트렌드

- 요일별/시간별 채팅분포도(붉은색에 가까울수록 횟수가 많음)

Viewer - 날짜별


- 기간필터

- 날짜별 사용자 채팅수 & 사용자 채팅 분포

- 해당 기간동안의 모든 채팅내역을 카카오톡처럼 나타냄
